# Markdown

一種輕量級標記語言，用於寫作，語法全由標點符號所組成，目標是實現「易讀易寫」。文檔後綴加上 *.md* 或 *.markdown*。

只涵蓋純文字可以涵蓋的範圍。





## 標題

標題格式有兩種，[Setext](http://docutils.sourceforge.net/mirror/setext.html)和[atx](http://www.aaronsw.com/2002/atx/)形式。

#### Setext

使用 ```=``` 和 ```-``` 。

`=`：最高階標題

`-`：第二階標題

```
This is an H1
=============

This is an H2
-------------
```

#### atx

使用 ```#``` 。

在行首插入1到6個 `#` ，各對應到標題1到6階

```
# This is an H1

## This is an H2

###### This is an H6
```



## 段落

段落是以兩個空白 ` spec` 和 `enter`

```
line 1  `spec``spec``enter` 
line 2
```

或是兩句中以一個空行

```
line1

line2
```



## 字體

#### *斜體*

```
*斜體*
_斜體_
```

#### **粗體**

```
**粗體**
__粗體__
```

#### ___粗斜體___

```
***粗斜體***
___粗斜體___
```



## 分隔線



***

三個以上的 `-` 或 `*` 

```
---
***
-----
*****
```



## 刪除線

#### ~~刪除線~~

在文字的兩端加上兩個 `~`

```
~~刪除線~~
```


## 列表

分成有序列表及無序列表。

####  無序列表 

使用 `*`或 `+`或 `-`

* 無序列表 1
* 無序列表 2
* 無序列表 3

```
* 無序列表 1
* 無序列表 2
* 無序列表 3

+ 無序列表 1
+ 無序列表 2
+ 無序列表 3

- 無序列表 1
- 無序列表 2
- 無序列表 3
```



#### 有序列表

在數字後加上`.`

1. 有序列表 1

2. 有序列表 2
3. 有序列表 3

```
1. 有序列表 1
2. 有序列表 2
3. 有序列表 3
```



## 子列表

在列表中換行使用`tab`。

- 主列表
  1. 子列表
     + 子列表
     + 子列表
  2. 子列表
- 主列表
  - 子列表
    - 子列表
      1. 子列表
      2. 子列表
